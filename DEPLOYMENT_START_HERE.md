## 🎯 YOU NEED TO DO (3 STEPS)

### Step 1: Render Dashboard (2 min)
Add 3 environment variables:
- `FRONTEND_URL=https://skinlytics-lyart.vercel.app`
- `CORS_ORIGINS=https://skinlytics-lyart.vercel.app,http://localhost:5173,http://localhost:3000`
- `ENVIRONMENT=production`

**Then wait 2-3 minutes**

### Step 2: Vercel Dashboard (1 min)
Add 1 environment variable:
- `VITE_API_URL=https://skinlytics-rdk0.onrender.com`

**Then redeploy**

### Step 3: Test (1 min)
- Open frontend: https://skinlytics-lyart.vercel.app/
- Fill form
- Should see results!

**Total: 5-10 minutes**
